GitHub

La première étape est de créer un dépôt GitHub

Le dépôt GitHub va nous permettre de stocker et de gérer le code des développeurs. Il permet également de partager le code avec d'autres développeurs aini que de suivre les modifications qui lui sont apportées

GitHub apporte deux éléments essentiels pour les développeurs : un accès au Git et un contrôle des versions de leur code

1- Donc première étape pour créer son site : créer un compte Github puis créer un dépôt

2- Création d'un répertoire

3- Télécharger Github Desktop et l'utliser pour cloner votre projet en cliquant sur "Set up in Desktop"

Une fois que cela est fait, je vous invite à apprendre les bases de l'HTML puis de regarder comment intégrer un template à votre site.

Intégrer un template

Le template est un mécanisme utilisé pour stocker du contenu HTML qui ne doit pas être affiché lors du chargement de la page mais qui peut être instancié et affiché par la suite grâce à un script JavaScript.

Il permet en autre de modéliser son site via des sources déjà existantes

Attention! Très compliqué de télécharger le template de votre choix **sarcasme. Pour le faire, il va falloir aller sur internet et choisir celui qui vous plaît le plus. Je vous conseille de naviguer sur le site Boostrap pour faire votre choix.

Une fois choisi, allez dans votre répertoire local et remplacez/ajoutez les éléments du template téléchargé

En tant que novice, vous avez désormais toutes les clès pour réussir votre site de façon simple et à votre goût. A vos créations!

Netlify

Netlify permet tout de simplement de déployer votre site sur le net à partir de votre dépôt GitHub

1- Créez vous un compte Netlify puis "importer votre projet"

2- Choisissez comme fournisseur "GitHub", selectionnez votre dépôt puis déployez-le!

Votre site est désormais en ligne!
